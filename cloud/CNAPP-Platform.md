# EMETRIX Sentinel XDR Cloud Security Platform (CNAPP)

## Executive Summary

The Cloud Security Platform provides Cloud Native Application Protection Platform (CNAPP) capabilities that unify CSPM, CWPP, CIEM, Kubernetes Security, IaC scanning, container runtime protection, vulnerability management, AI-powered cloud analytics, and automated cloud response.

The platform continuously protects AWS, Azure, Google Cloud, Kubernetes, Docker, Serverless workloads, and hybrid cloud infrastructures.
